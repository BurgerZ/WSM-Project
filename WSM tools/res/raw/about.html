Install:

  1. Launch WSM tools
  2. Go to the  Settings — Manage WSM
  3. Press Install button
  4. Reboot

After ROM update:

  1. Every time you update your ROM you need to reinstall WSM binaries, because app_process is replaced by the original file.

Delete (in case you get into a boot loop):

  1. Reboot to Recovery
  2. Install Xposed-Disabler-Recovery.zip (this file was copied to your SD card when you install WSM)
Or:
  1. Create a file /data/data/pro.burgerz.wsm.manager/conf/disabled, which causes Xposed to be bypassed.
